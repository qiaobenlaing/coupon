惠圈商户合作协议

惠圈是中国工商银行股份有限公司浙江省分行联合杭州新利软件有限公司共同打造的互惠互利圈子平台。

本协议由缔约双方在自愿、平等、公平及诚实信用原则的基础上，根据《中华人民共和国合同法》等相关法律、法规的规定，经友好协商缔结。

一、相关定义及解释

1.1 “惠圈”平台：致力于打造真实、可靠、便捷的本地化生活服务平台，基于消费者的地理位置，随时随地为消费者提供最新最全的本地化活动优惠信息以及相关服务。精准位置的券分享使消费者随时随地发现身边的实惠体验。惠圈以电子会员卡的形式打通顾客与商家的沟通渠道，建立顾客与商家的关联度，让消费者可随时、随地、简单、便捷地享受会员优享待遇，提升会员对商家品牌的认知度。

1.2 “实体合作商户”：指与 “惠圈” 签订《惠圈商户合作协议书》、合作开展惠圈相关服务，并接受使用银行卡或者POS机完成资金结算的企事业单位、个体工商户或其他组织。

1.3 商家注册：商家注册指欲成为“惠圈”第三方经营者的商家，依据“惠圈”注册流程和使用要求完成在线信息提交，经“惠圈”审核同意后，开展经营活动；

1.4 商家使用：指第三方经营者完成商家注册，通过资质审核且满足本协议服务开通条件后成为“惠圈”第三方经营者的过程；本协议中的商家指本协议缔约方中的“乙方”。

1.5 连锁店门店：对于甲方存在连锁店的情况，乙方将按每个独立门店进行管理。

1.6 本协议所称优惠券是指甲方通过乙方服务平台向消费者（“用户”）提供的服务或产品。

二、协议内容及生效

2.1 本协议内容包括协议正文及所有惠圈已经发布的或将来可能发布的各类规则。所有规则为协议不可分割的一部分，与协议正文具有同等法律效力。

2.2 商户在使用惠圈提供的各项服务的同时，承诺接受并遵守各项相关规则的规定。惠圈有权根据需要不时地制定、修改本协议或各类规则，如本协议有任何变更，惠圈将在第一时间经过客户经理通知商户，任何修订和新规则一经公示，商户在七日内如未以书面形式提出相关意见，则新规则即时生效，并成为本协议的一部分。登录或继续使用服务将表示商户接受经修订的协议。除另行明确声明外，任何使服务范围扩大或功能增强的新内容均受本协议约束。

2.3 甲方应保证其提供的信息内容真实、准确、合法，符合中国相关法律、法规及政策的规定，亦不会损害任何第三方的合法权益。乙方有权对甲方提供的相关信息内容进行合理审查，对不符合法律、法规及政策或乙方有理由相信发布该信息或执行该项推广活动将给乙方带来不利影响的内容，乙方有权要求甲方进行修改。在乙方对相关内容审核通过前，乙方有权拒绝进行相关的发布和推广。

2.4 在下述情况下，乙方有权部分或全部终止推广并不向甲方承担任何责任：推广所必需的系统定期和紧急维护；乙方网站的重新上线或重新设计或网站功能的扩展；电信服务不可预知的事故、暂停或中断、电信网络连接失败、黑客袭击、分裂或第三方的非法入侵以及其他不能归因于乙方的电信故障；或其他乙方合理认为应当终止推广的事件。

2.5 提供优惠券服务：用户通过乙方网站领取户优惠券后，有权要求并享受甲方提供的优惠券上所载明的产品或服务。

2.6 为方便审核，请合作商户协助提供以下申请资料，所提供的相关申请资料复印件需加盖单位公章或财务专用章。

  • 有效的营业执照；
  • 税务登记证件或相关纳税证明；
  • 银行开户许可证；
  • 法定代表人或负责人有效身份证件(如属非法定代表人或非负责人经办的，还应要求商户出具授权经办人有效身份证件及法人委托授权书原件）；
  • 个体户需提供证明：个体工商户营业执照、税务登记证、个体户身份证或个体工商户拥有同名个人银行结算账户的合法证明文件；
  • 其他详细资料见本协议附件。

三、知识产权及保密条款

3.1 由乙方提供的摄影图片、文字等资料的知识产权归乙方所有，如甲方选择由乙方为其提供优惠券设计，则该设计的内容、布局、结构等方面的设计成果的知识产权归乙方所有；甲方应保证由乙方提供的图片、文字及设计等资料仅限用于本协议项下合作之目的，不得用于其它任何用途。

3.2 在双方执行本协议的过程中，对于所知悉的对方的一切商业秘密、其它秘密信息及本协议的存在及其内容，双方均负有保密义务，未经对方书面许可，不得自己使用、允许他人使用或向任何第三方透露。如一方提出要求，另一方应将载有对方保密信息的文件、资料或软件等，按要求归还、予以销毁，或进行其他处置，并不得继续使用、允许他人使用或向第三方披露这些保密信息。

3.3 甲乙双方就用户的信息权益，同意并保证遵守以下约定：

3.3.1 甲方保证其由乙方处获得的用户所有信息仅限用于本次交易，不得做任何其它用途或向任何第三方披露，否则应承担因此造成的一切损失，包括但不限于赔偿乙方及其它任何第三方因此遭受的经济和其它方面的损失；

3.3.2 乙方保留在必要时进行诉讼的权利。

3.4 乙方向甲方提供的有关用户的信息（包括但不限于：用户姓名、手机号、生日、出生地等）均为保密信息并属于乙方所有，甲方承诺仅用于兑换本协议项下优惠券之唯一目的，不得用于其他任何用途或向其他任何第三方进行披露，也不得利用上述用户之信息从事任何非法活动或从事任何可能对乙方声誉或乙方利益造成损害的活动。

3.5 双方的保密义务不以本协议的有效期为限，本协议终止后三年内双方仍互相负有保密义务。

四、陈述与保证

甲方声明、保证如下：

4.1是合法设立并有效存续的商业实体；

4.2有资格从事本合同项下之交易，并且该交易符合其经营范围的规定；

4.3有能力履行其于本合同项下之义务；

4.4并非清算、解散或破产程序之主体；

4.5如果发生可能会影响到双方运营或影响到本协议履行的任何变更，一方应立即将该变更书面通知另一方；

4.6尽最大努力，以合理的技能和谨慎全面地履行本合同规定的所有义务。

五、违约责任及争议解决

5.1若因一方违约给对方造成不良社会影响或经济损失，则另一方有权追究违约方违约责任，要求其消除影响、承担赔偿责任。

5.2与本协议有关的任何纠纷或争议应通过双方友好协商解决。如不能协商解决，任何一方可提交至乙方所在地的法院解决。本协议受中华人民共和国大陆法律管辖。

5.3本协议包含双方关于此事项的全部约定。双方在此之前达成的任何口头或书面的协议或约定，如果与本协议冲突，应以本协议内容为准。本协议的任何修改须以书面方式做出并经双方书面确认后才能生效。

5.4本协议中未尽事宜双方可协商签订补充协议并作为本协议的附件，与本协议具有同等法律效力。

六、有效期及终止

6.1本协议于双方签字盖章之日起生效。

6.2如一方违反本协议约定导致本协议无法履行，另一方在向违约方发出书面通知后有权终止本协议。

6.3本协议的终止或提前终止不影响双方截至终止之日已发生的权利或义务；不影响甲方已向用户出具的优惠券的有效性，甲方应依照已发优惠券使用规则向用户提供相应的服务或产品。

七、不可抗力

7.1“不可抗力”是指本合同双方不可预见、不可控制并不可避免的事件。该事件包括但不限于电信服务的不可预知的事故、暂停或中断、电信网连接失败、政府行为、自然灾害、战争、黑客袭击、分裂或第三方的非法入侵或任何其他类似事件。

7.2遭受不可抗力事件的一方可暂行中止履行本合同项下的义务直至不可抗力事件的影响消除为止，并且无需为此而承担违约责任；但遭受不可抗力事件的一方应尽最大努力及时采取合理措施，减少由不可抗力事件引起的损失；否则应承担相应的责任。
